# WebTracer

Lightweight tracing/visualization project.

Quick start

1. Initialize repository (already done locally):
   - `git init`
2. Install dependencies (if any):
   - `npm install` or `yarn`
3. Run in browser by opening `index.html`.

To create a remote GitHub repository and push the initial commit, use the GitHub CLI:

```bash
gh repo create YOUR_GITHUB_USERNAME/WebTracer --public --source=. --remote=origin --push -y
```

If you prefer the web UI, create a repo on GitHub and add the `origin` remote:

```bash
git remote add origin git@github.com:YOUR_GITHUB_USERNAME/WebTracer.git
git push -u origin main
```
